# 🌦️ Weather app (Python)
### 📌This is a simple Python weather application that fetches real-time weather data using the OpenWeatherMap API. The user enters a city name, and the app displays temperature and weather condition.
___

## 🚀 Features
- Get real-time weather data 🌍
- User input for any city
- shows temperature in Celsius 🌡️
- Displays weather condition (clear, cloudy, etc.)
- Handles invalid city input
___
## 🛠️ Technologies Used
- Python
- request library
- OpenWeatherMap API
